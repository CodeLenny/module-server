. (Promise) delayUntil(poll = 25, test)

Delays until some test is true. Currently uses polling.

Parameters:

  • poll ( Integer ) — Optional how often the test should be checked, in milliseconds. Defaults to 25ms.
  • test ( Function<Boolean> ) — function to determine if progress should continue. Resolves the promise as soon as the test returns true.

Returns:

  • ( Promise ) — Resolves after the test returns true.
