# (ModuleTest) onit(name, handler, timeout, cb)

Provides a test that will be triggered by client calls to TestResponse.emit.

Parameters:

  • name ( String ) — the name to give to Mocha's it
  • handler ( String ) — the event that will be called by TestResponse.emit
  • timeout ( Integer ) — Optional changes Mocha's timeout. Defaults to 2000ms.
  • cb ( Function ) — called with the data given to TestResponse.emit, to provide chai testing.

Returns:

  • ModuleTest

# (ModuleTest) on(handler)

Listens to a handler. Returns an object containing a method to define tests.
