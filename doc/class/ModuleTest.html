Constructor Details

# (void) constructor(describe, it, before, after, chai, describeName)

Initializes a new testing environment for a series of tests (it blocks) inside a single describe block.

Parameters:

  • describe ( Mocha.describe ) — a copy of the describe function from Mocha
  • it ( Mocha.it ) — a copy of the it function from Mocha
  • before ( Mocha.before ) — a copy of the before function from Mocha
  • after ( Mocha.after ) — a copy of the after function from Mocha
  • chai ( Chai ) — testing instance of Chai
  • describeName ( String ) — a name to use for the describe block.

# (ModuleTest) onit(name, handler, timeout, cb)

Provides a test that will be triggered by client calls to TestResponse.emit.

Parameters:

  • name ( String ) — the name to give to Mocha's it
  • handler ( String ) — the event that will be called by TestResponse.emit
  • timeout ( Integer ) — Optional changes Mocha's timeout. Defaults to 2000ms.
  • cb ( Function ) — called with the data given to TestResponse.emit, to provide chai testing.

  • ModuleTest
