Class: ModuleServer

Defined in: src/ModuleServer.coffee

Overview

Connect plugin to serve modules.

Property Summary

(Object) _loadedPackages
Stores modules that have been loaded as {name, main, paths}, where name is the client-side module name defined, main is a client-side path to the main script, and paths is an object of {script: location} of additional files to serve.

Instance Method Details

# (void) load(name, path)

Loads a client module, and serves it under the provided name. Blocks modules from being loaded multiple times.

Valid paths:

Type Example
Absolute path: load("ServerConnection", "/usr/local/serverconnection/client/")
NPM package: load("URLParser", "@codelenny/url-parser")
Directory in NPM package: load("QueryParser", "@codelenny/url-parser/query")
Relative path: load("Logging", "../../advanced-logging/")

Parameters:

  • name ( String ) — module name to provide to the client
  • path ( String ) — location of module source files

# (void) correctPaths(name, paths)

Modules can define paths to files under package.json that will be provided to RequireJS. Substitute locations to public files via $PUBLIC, and remove trailing .js extensions.

For example, to provide a local copy of a library, underscore.js located in the public directory of a module loaded via load("UserCounter", "@codelenny/user-counter"):

// in package.json
{ ..., "com.codelenny.paths": {
    "_": "$PUBLIC/underscore",
    "jquery": "https://cdnjs.cloudflare.com/ajax/libs/jquery/3.1.0/core.js" }
}

define(["UserCounter/_", "UserCounter/jquery"], function(_, $) { // _ and $ can be used
});

Parameters:

  • name ( String ) — module name to provide to the client
  • paths ( Object<String, String> ) — an object of url locations to serve, and locations of files

# (String) findPath(path)

Locates a node module via require.resolve, then stripping the class. See {ModuleServer.load} for example paths.

Parameters:

  • path ( String ) — either an absolute file path, or a path to resolve.

Returns:

  • ( String ) — a full path to the module required.
