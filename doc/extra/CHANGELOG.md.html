0.4.1 - 2016-10-24

DEPRECIATED Contains outdated assets. Use 0.4.2 instead.

Added

  • ModuleTest#chrome logs the URL if debug mode is on, in case the window doesn't automatically open.

Changed

  • ModuleTest#startPhantom now uses page.on instead of page.property for listening, allowing other scripts to also listen.

0.4.0 - 2016-10-06

Breaking Changes

  • Using layers of routing (express.Router) to increase performance. To do so, ModuleServer#finalize() was added as a necessary method, to be put after all .load() calls. ModuleServer will break on systems that lack #finalize().
